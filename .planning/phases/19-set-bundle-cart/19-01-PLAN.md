# Plan 19-01 — DB Migration + Cart Service + Cart Store

## Goal

Add `cart_bundles` and `order_bundle_items` tables to the DB, extend `CartService` with bundle operations, and add bundle state to the cart store.

## Context (from 19-CONTEXT.md)

- `cart_bundles`: separate table, mirrors `cart_items`, FK → `sets`, quantity-based, UNIQUE(cart_id, set_code)
- `order_bundle_items`: separate table, mirrors `order_items`, snapshots set_name + price_at_purchase
- Cart price: live `sets.price` (no snapshot in cart); order snapshotted
- Merge on login: `cart_bundles` rows must be merged alongside `cart_items` (upsert by set_code, sum quantities)

## Tasks

### Task 1 — DB Migration

**File:** `supabase/migrations/20260614000002_cart_bundles.sql`

```sql
-- cart_bundles: one row per set per cart (quantity-based)
CREATE TABLE public.cart_bundles (
  id          UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  cart_id     UUID NOT NULL REFERENCES public.carts(id) ON DELETE CASCADE,
  set_code    TEXT NOT NULL REFERENCES public.sets(set_code) ON DELETE CASCADE,
  quantity    INTEGER NOT NULL DEFAULT 1 CHECK (quantity >= 1),
  added_at    TIMESTAMPTZ NOT NULL DEFAULT now(),
  updated_at  TIMESTAMPTZ NOT NULL DEFAULT now(),
  UNIQUE (cart_id, set_code)
);

-- RLS: same pattern as cart_items
ALTER TABLE public.cart_bundles ENABLE ROW LEVEL SECURITY;

-- Users can read/write their own cart bundles (via cart ownership)
CREATE POLICY "cart_bundles_user_access" ON public.cart_bundles
  USING (
    cart_id IN (
      SELECT id FROM public.carts
      WHERE user_id = auth.uid()
         OR guest_id = current_setting('app.guest_id', true)
    )
  );

-- order_bundle_items: permanent record of set bundles in an order
CREATE TABLE public.order_bundle_items (
  id                UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  order_id          UUID NOT NULL REFERENCES public.orders(id) ON DELETE CASCADE,
  set_code          TEXT NOT NULL,  -- denormalized (set may be renamed/deleted)
  set_name          TEXT NOT NULL,  -- snapshot at checkout
  quantity          INTEGER NOT NULL DEFAULT 1 CHECK (quantity >= 1),
  price_at_purchase NUMERIC(10, 2) NOT NULL,
  created_at        TIMESTAMPTZ NOT NULL DEFAULT now()
);

-- RLS: users can read their own order bundle items
ALTER TABLE public.order_bundle_items ENABLE ROW LEVEL SECURITY;

CREATE POLICY "order_bundle_items_user_read" ON public.order_bundle_items
  FOR SELECT USING (
    order_id IN (
      SELECT id FROM public.orders WHERE user_id = auth.uid()
    )
  );

-- Service role has full access (for admin operations)
-- (service role bypasses RLS by default)

-- updated_at trigger for cart_bundles
CREATE TRIGGER cart_bundles_updated_at
  BEFORE UPDATE ON public.cart_bundles
  FOR EACH ROW EXECUTE FUNCTION moddatetime(updated_at);
```

**Verify:** `npx supabase db diff` shows new tables; `npm run db:generate:remote` after applying.

### Task 2 — CartService Bundle Methods

**File:** `src/lib/server/cart-service.ts`

Add 4 new methods to the `CartService` class:

#### `addBundle(cartId, setCode, quantity)`
```
1. Fetch set to confirm it exists and get live price
2. Upsert into cart_bundles (ON CONFLICT (cart_id, set_code) DO UPDATE SET quantity = quantity + excluded.quantity)
3. Return updated bundles list
```

#### `removeBundleItem(cartId, bundleId)`
```
1. DELETE from cart_bundles WHERE id = bundleId AND cart_id = cartId
2. Return updated bundles list
```

#### `updateBundleQuantity(cartId, bundleId, quantity)`
```
- quantity <= 0 → remove
- quantity > 0 → UPDATE
```

#### `getCartBundles(cartId)` → `CartBundle[]`
```ts
type CartBundle = {
  id: string
  cart_id: string
  set_code: string
  quantity: number
  added_at: string
  set: { set_code: string; set_name: string; price: number | null }
}
```
```
SELECT cart_bundles.*, sets(set_code, set_name, price)
WHERE cart_id = cartId
ORDER BY added_at ASC
```

#### Extend `getCartWithItems` (or add overload)
Return bundles alongside items:
```ts
async getCartWithItems(cartId): Cart & { items: ...[]; bundles: CartBundle[] }
```

#### Extend `mergeGuestCart` — also merge `cart_bundles`
After merging `cart_items`, merge `cart_bundles`:
- For each guest bundle: upsert into user cart (increment quantity if set_code already present)
- Delete guest cart_bundles

#### Extend `clearCart` — also delete `cart_bundles`
```
DELETE FROM cart_bundles WHERE cart_id = cartId
```
