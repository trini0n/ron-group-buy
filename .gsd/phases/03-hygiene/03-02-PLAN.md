  <action>
    First read `src/lib/server/logger.ts` to understand the exported interface.

    If the logger doesn't have a `.error(message, ...args)` method compatible with `console.error`, add one:
    ```typescript
    export const logger = {
      error: (msg: string, ...args: unknown[]) => console.error(`[ERROR] ${msg}`, ...args),
      warn: (msg: string, ...args: unknown[]) => console.warn(`[WARN] ${msg}`, ...args),
      info: (msg: string, ...args: unknown[]) => console.log(`[INFO] ${msg}`, ...args),
    };
    ```

    Then in each service file, find `console.error(...)` calls and replace with `logger.error(...)`. Add the import at the top:
    ```typescript
    import { logger } from '$lib/server/logger';
    ```

    Focus on `notifications/service.ts`, `cart-service.ts`, `export-builder.ts`, `gphoto-converter.ts`. Do NOT touch client-side files or `+page.svelte` files.
