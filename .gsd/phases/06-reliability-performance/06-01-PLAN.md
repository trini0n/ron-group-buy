<objective>
Two independent reliability improvements:

1. **LRU cache for deck import** — replace the unbounded `new Map()` in the deck import API with `LRUCache` (already in `package.json` as `lru-cache@^11`). The plain Map grows without bound across server lifetime; LRU caps at 100 entries and handles TTL expiration natively, eliminating the manual `getCached`/`setCache` timestamp dance.

2. **Eliminate duplicate AdminClient creation** — `isAdmin()` currently calls `createAdminClient()` internally, so every caller that already has an admin client (e.g., `isAdminRequest()`, `requireAdmin()`, every `verifyAdmin()` helper in route files) creates a second throwaway connection. Adding an optional `adminClient` parameter to `isAdmin()` allows callers to pass their existing client.

Purpose: Caps memory growth from unbounded Map; reduces Supabase connection churn from O(2N) to O(N) per admin request.  
Output: LRUCache import + usage in deck handler; isAdmin() signature change + updated callers in admin.ts.
</objective>
