# Plan 04-01: DB Type Regen + Cast Removal

## Objective

Regenerate Supabase database types and remove all `@ts-ignore` / `as any` casts that were patching missing type definitions.

## Context

Several files contain `@ts-ignore` annotations and `as any` casts that were added as temporary workarounds because the generated `database.types.ts` was missing the `addresses.phone_number`, `orders.shipping_phone_number`, and `card_type_pricing` table definitions. After regenerating types, these workarounds become unnecessary and must be removed so `npm run check` passes clean.

```bash
npm run db:generate
```
